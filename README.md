# angular-firebase-crud

Projeto com o objetivo de aprender fazer a utilização do framework Angular e o Firebase
+ [X] Layout
+ [X] Comunicação firebase
+ [X] Firebase authentication
+ [X] Operações Crud
+ [ ] Revisões gerais 

<h5> Instruções: </h5>

- Ambiente: Node js, Angular 8
- npm install
- ng serve
- ng build --prod
- firebase deploy
